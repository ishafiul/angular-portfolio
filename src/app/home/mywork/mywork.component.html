3X - Multiplayer Game

3X is a game where two players will play in a single lobby and the main goal is to pic perfect 3 numbers where the multiplication of these 3 numbers is equal to the given result. Here users can create new games, can invite friends, players can chat inside game lobby or globally. The main goal was for this project to know how a WebSocket connection work, and basic docker learning.

I created this when I started my freelancing carer and I know nothing about laravel and other frameworks. LOL.
